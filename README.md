# letsencrypt

`letsencrypt` will provide the dependencies to use [dehydrated](https://github.com/lukas2511/dehydrated), a Letsencrypt ACME client.

## Compatibility

This bundle has been tested on the following systems:

| OS          | `[x]` |
| ----------- | ----- |
| Fedora 24   | `[x]` |
| Fedberry 24 | `[ ]` |
